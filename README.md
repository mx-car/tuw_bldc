# tuw_bldc
Brushless DC Motor Cotnroler and PCB Layouts

To build and upload the code, PlatformIO is being used. To get PlatformIo:

[PlatformIO Installation Guide](https://docs.platformio.org/en/latest/installation.html#virtual-environment) 

After getting this tool, you can just open the project folder in /Software wiht any IDE supported by PlatformIO. The current setup is for Clion so if any other IDE is being used refer to PlatformIO [integration guide](https://docs.platformio.org/en/latest/ide.html#desktop-ide). This step is also required for Clion.

After that the upload process is straightforward.
